# gv2_esp32_sd

GV2 + ESP32-S3 stack for saving detection images to SD.

- **Part 1:** Adapted GV2 image (from or alongside `gv2_firmware` output.img).
- **Part 2:** ESP32-S3 firmware (this PlatformIO project) for UART and SD capture.

See `notes/himax gv2 sd jpeg image storage after detection.md` and `notes/_hardware_stack.md` for hardware and workflow.
