The VespAI GitHub repository is now accessible via the original link provided in the publication (https://github.com/andrw3000/vespai). This includes setup instructions, manuals, and requirements published under a CC BY-NC-SA 4.0 license, and model code conveyed under a AGPL-3.0 license.
The system is still in development, and consequently should be considered as a research tool, rather than as a finished product. As such, it needs to be compiled from the relevant packages detailed in ‘requirements.txt’, and deployed through the Python command line.
To achieve the performance described in the paper, the hardware setup needs to match precisely with the test setup detailed in the associated publication.
We are currently developing a more user-friendly version of VespAI, including a GUI and integrated hardware.
If you are interested in further collaboration, we are also open to discussing possibilities for adaptation to specific research use-cases.
